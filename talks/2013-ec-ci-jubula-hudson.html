start & test AUT - standard workflow

1. start autagent
on test machine

2. start testexec
loads tests from database

3. testexec » autagent: "start & wait for AUT"
via AUT configuration

4. test suite execution
documents all events / steps

5. testexec provides test result
in database & file-system
